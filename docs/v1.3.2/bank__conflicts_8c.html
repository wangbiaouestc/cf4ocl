Detailed Description

Bank conflicts test example. Control the level of conflicts using the stride -s parameter.

The frequency of bank conflicts can be increased by doubling the stride -s parameter, e.g. 1, 2, 4, 16, 32. The maximum number of conflicts is obtained with s=16 or s=32, depending if the GPU has 16 or 32 banks of local memory.
